Bleu Sample
===========

This is sample code designed for use with [Bleu Stations](http://www.twocanoes.com/bleu)
by Twocanoes. Some functions may work with any standard iBeacon.

Bleu Locate
-----------
Bleu Locate demonstrates how to use CoreLocation to for region monitoring and
ranging.Sample code for showing different web views based on proximity to an iBeacon.


Bleu Settings
-------------
Bleu Settings uses libbleu to discover Bleu Stations in range and to read and
write the settings of the station. This project does not include the libbleu
files. Those files must be copied to the source file for the project to compile.
